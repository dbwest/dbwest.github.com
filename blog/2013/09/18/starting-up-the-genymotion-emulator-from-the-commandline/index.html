Starting Up the GenyMotion Emulator From the Commandline
Sep 18th, 2013

This is really useful for CI. In your Jenkins or Hudson jobs, you will want to be able to script emulator startup and shutdown.

Howto

First: VBoxManage list vms to see all your VMs

Then: /Applications/Genymotion.app/Contents/MacOS/player --vm-name "xxxx" (xxxx is virtual device name)
